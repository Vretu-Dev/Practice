# Practice
This is a custom mode called “practice.” Instead of a standard round, players respawn as a tutorial on a tower and have access to commands that allow them to create duels on a limited map. This allows them to practice their aim and improve their skills.
### Required permission
- practice.all
- practice.admin (only for admin commands)
### Arena List:
- HczCheckpointA
- HczCheckpointB
- LczCheckpointA
- LczCheckpointB
- GateA
- GateB
- Glassroom
- Chamber049
### Modes:
- 1v1
- 2v2
### Types:
- Classic (every player has same random loadout)
- RandomClassic (every player has different random loadout)
- Jailbird
- ParticleDisruptor
